# vdrum-game
----

VDrum é um futuro jogo de ritmo, estilo guitar-hero.
Neste projeto, a base do <a href="https://github.com/hayukimori/RealDrum-Ursina-Version">RealDrum-Ursina-Version</a> foi usada.

Requisitos
--
- Ursina: `pip install ursina`
- configparser
